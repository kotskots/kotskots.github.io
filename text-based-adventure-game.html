Text-Based Adventure Game

Ανέπτυξα ένα text-based adventure game σε Java, υλοποιώντας διάφορες λειτουργίες,όπως σύστημα αποθήκευσης/φόρτωσης, μάχης, αλληλεπίδρασης με αντικείμενα, και πολλά άλλα.

Ευθύνες

  • Σχεδιασμός και υλοποίηση της δομής του παιχνιδιού.
  • Δημιουργία κειμένου και περιγραφών για το περιβάλλον και τα αντικείμενα.
  • Υλοποίηση συστήματος αποθήκευσης/φόρτωσης για πρόοδο παιχνιδιού.
  • Ανάπτυξη συστήματος μάχης με διαφορετικές επιλογές και στρατηγικές.
  • Δημιουργία λογικής για αλληλεπίδραση με αντικείμενα και το περιβάλλον.
  • Εντοπισμός και διόρθωση σφαλμάτων.

Δεξιότητες

  • Ανάπτυξη Java
  • Σχεδιασμός παιχνιδιών
  • Δημιουργία κειμένου
  • Λογική προγραμματισμού
  • Αντικειμενοστραφής Προγραμματισμός
