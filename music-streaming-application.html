Music Streaming Application

Συμμετείχα στην ανάπτυξη μίας εφαρμογής Android, η οποία λειτουργεί ως music streaming application. Συνεισέφερα στην υλοποίηση βασικών λειτουργιών, όπως η σύνδεση, η αναζήτηση και η εμφάνιση πληροφοριών για μουσική.

Ευθύνες

  • Ανάπτυξη του interface για το signup και το login.
  • Υλοποίηση της μεθόδου fetchTopAlbums για εμφάνιση πληροφοριών για τα topalbums.
  • Δημιουργία RecyclerView για εμφάνιση των top albums.
  • Συγγραφή tests για τις κλάσεις που αφορούν τα πιο αγαπημένα άλμπουμ.
  • Υλοποίηση search για albums.
  • Αποθήκευση στη βάση δεδομένων των κομματιών, καλλιτεχνών και άλμπουμ πουο χρήστης liked.

Δεξιότητες

  • Ανάπτυξη Android UI
  • Κατανόηση και χρήση APIs
  • Υλοποίηση RecyclerView
  • Συγγραφή tests
  • Firebase
