Επαγγελματική Περίληψη

Απόφοιτος Μηχανικός Πληροφορικής με εξειδίκευση στην ανάπτυξη ιστοσελίδων WordPress και στη δημιουργία responsive και φιλικών προς τον χρήστη web εμπειριών. Διαθέτω πρακτική εμπειρία στην κατασκευή και παραμετροποίηση ιστοσελίδων χρησιμοποιώντας WordPress, Elementor, HTML και CSS, με έμφαση στη λειτουργικότητα, την αισθητική και την εμπειρία χρήστη.

Έχω εργαστεί σε digital agency περιβάλλον αναπτύσσοντας ιστοσελίδες για πελάτες, εφαρμόζοντας custom styling, βελτιστοποιήσεις UI/UX και βασικές πρακτικές SEO. Διαθέτω εμπειρία στην ενσωμάτωση plugins, στη διαμόρφωση layouts με page builders και στη δημιουργία responsive ιστοσελίδων για διαφορετικές συσκευές. Άτομο με προσοχή στη λεπτομέρεια και αναλυτική σκέψη, με εμπειρία συνεργασίας σε ομάδες για την ανάπτυξη σύγχρονων και αποδοτικών web λύσεων.
